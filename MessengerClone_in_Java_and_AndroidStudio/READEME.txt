Θα πρέπει να τρέξουμε τους Brokers μέσα από κάποιο IDE καθώς για τα path των αρχείων χρησιμοποιούμε relative paths (μέσω του workspace dir)

Για να τρέξει η εφαρμογή θα πρέπει πρώτα να μορφοποιήσουμε κατάλληλα το αρχείο InitBrokers.txt στον φάκελο InitBrokers.

1)Θα πρέπει μέσα,στην πρώτη γραμμή να δώσουμε τον αριθμό των brokers που επιθυμούμε να έχουμε και στη συνέχεια από κάτω οι ακόλουθες γραμμές να έχουν την μορφή:
	(numberOfBroker,IP,Port) 
 όπου το numberOfBroker θα πρέπει να ξεκινά από 1 και τα IP και Port να είναι τα στοιχεία του υπολογιστή που θα τρέξει τον Broker.
	
(Μέσα στο αρχείο μπορούμε να έχουμε και περισσότερους Brokers από αυτούς που θα τρέξουμε δηλαδή ο αριθμός των broker στην πρώτη γραμμή του αρχείου μπορεί να είναι μικρότερος από τους καταγεγραμμένους Broker 
 στο αρχείο, αλλά δεν μπορεί να είναι μεγαλύτερος)

2)Μετά κάτω από το σύμβολο % είναι τα ονόματα των Topics.Εκεί απλά βάζουμε σε κάθε γραμμή το όνομα του Topic που θέλουμε και δεν υπάρχει περιορισμός στον αριθμό τους.

3)Για να τρέξει το πρόγραμμα αρχικά πρέπει να ανοίξουμε όλους τους Brokers που ορίσαμε στην αρχή του αρχείου.Στην αρχή της εκτέλεσης του Broker πρέπει να δώσουμε έναν αριθμό από 1 έως τον αριθμό που ορίσαμε.
Δεν πρέπει να δώσουμε 2 φορές τον ίδιο αριθμό αλλιώς θα υπάρχει πρόβλημα στην επικοινωνία των broker.

4)Θα πρέπει να αλλάξουμε και την ip στον consumer στο κομμάτι του android για να μπορεί να συνδεθεί με έναν broker για να λειτουργεί η εφαρμογή.



ΑΛΛΑΓΕΣ ΣΕ ΣΧΕΣΗ ΜΕ ΤΗΝ ΠΡΩΤΗ ΦΑΣΗ:

Διορθώθηκε το λάθος με τα chunks και πλέον η αναδόμηση του αρχείου γίνεται μόνο στον user, ο broker κρατάει κανονικά το αρχείο σε chunks.
Γίνεται το απαραίτητο synchonize για να μην βρεθούν δύο αρχεία μπλεγμένα μεταξύ τους όταν αποθηκεύονται στον broker.
Θυμάται το όνομα κάθε user που συνδέθηκε, έτσι αν αποσυνδεθεί κάποιος user δεν θα ξαναλάβει μετά μηνύματα που έχει δει.
